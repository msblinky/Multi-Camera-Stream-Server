# Multi-Camera-Stream-Server

How to install:
<br>
Clone the repository to the desired location on your hard disk.
<br>
How to run?
<br>
cd /path/to/your/folder
<br>
STREAM_PASSWORD=yourpassword python3 server.py 8080
<br>
PLEASE use a secure password and keep it safe.
<br>
Configure:
<br>
You get the configuration menu by opening this page and providing the password that only you know.
<br>
http://localhost:8080/?config=1
<br>
How to use:
<br>
Add 
<br>
http://localhost:8080 
<br>
to your OBS setup as a browser source.
<br>
Have fun streaming!
